Installation

Installing

Make sure, your hostname is added to the hostlist and you have the B.A.T.M.A.N. kernel module installed (some distros already ship it per default).
Then run this commands: 
wget "{{request.url_root}}f3lnet.sh" -O /usr/local/bin/f3lnet
chmod +x /usr/local/bin/f3lnet
f3lnet refresh

Configuration

If your Wifi-Device has the Identifier "wlan0", everything should work out of the Box. Otherweise you may set the variable 
WIFIDEV='\'
either as environmental, or in '/etc/f3lnet.conf'.

Starting

f3lnet start

Using

You sould be able to reach the other nodes via hostname.fn. See the debugging section for B.A.T.M.A.N. layer commands.

Stopping

f3lnet stop
